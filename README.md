## 🚀 Installation
  
```bash
# Repository clone
$ git clone https://github.com/ImGabreuw/books-api.git

# Go to the application directory
$ cd books-api

# Start database
$ docker-compose up -d

# Run application
$ go run main.go
```

Go to http://localhost:5000/api/v1/books/
